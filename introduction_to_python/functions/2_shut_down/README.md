# Shut Down

## Motivation
Keyword arguments allow caller to specify argument name with values so that the caller does not need to remember order of parameters. 
Boolean type- True/ False 
## Problem Description
Write a Python function `shut_down` that consumes a boolean value `x`.
If `x` is the boolean value `True` return the string `"SHUTDOWN"`, otherwise return the string `"SHUTDOWN ABORTED"`.
